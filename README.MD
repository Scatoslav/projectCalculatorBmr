# Калькулятор калорий и генератор меню

Приложение рассчитывает суточную норму калорий (BMR), подбирает БЖУ, генерирует меню на день или неделю и сохраняет данные пользователя в браузере.

## Используемые технологии

- HTML5
- CSS3
- JavaScript (ES Modules, `type="module"`)
- Browser API:
  - `localStorage` для сохранения формы, результата и меню
  - `window.print()` для печати/сохранения отчета в PDF через системный диалог браузера
  - DOM API (`querySelector`, `addEventListener`, `insertAdjacentHTML`)

## Структура проекта

- `1.html` — главная страница калькулятора и генерации меню
- `resipes.html` — база рецептов с фильтрацией
- `1.css` — стили интерфейса
- `js/main.js` — основной сценарий страницы
- `js/calculator.js`, `js/calculations.js` — расчеты калорий и БЖУ
- `js/generateMenuDay.js`, `js/generateMenuWeek.js` — генерация меню
- `js/menu.js` — база рецептов
- `js/saveData.js` — сохранение/восстановление данных
- `js/pdf.js` — формирование печатной версии для PDF
- `js/helpers.js` — вспомогательные функции

## Запуск

Проект не требует сборки и запускается как статический сайт:

1. Откройте `1.html` в браузере.
2. Пользуйтесь калькулятором и генератором меню.
